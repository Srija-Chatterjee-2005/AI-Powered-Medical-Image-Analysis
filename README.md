# **🏥 AI-Powered-Medical-Image-Analysis**
---

AI-powered medical imaging web app built using Flask and TensorFlow to detect pneumonia from chest X-rays. Features include dual-image comparison, confidence scoring, heatmap visualization, interactive charts, and a modern dashboard UI for real-time analysis.

## **📌 Overview**
---

This project presents an AI-based medical imaging system that analyzes chest X-ray images to detect Pneumonia. It simulates real-world healthcare workflows using deep learning and computer vision.

The system uses a trained CNN model to classify images and provides visual insights, heatmaps, and comparative analysis.
